# silounloadr
Functions to convey and prep data from KIPP Chicago's Silo data warehouse to an R session.

# .Renviron
Users will need to supply a `.Renviron` file in their home directory with 
the following variables set:

*  `SILO_DBNAME_NWEA`
*  `SILO_DBNAME_PS`
*  `SILO_DBNAME_ILL`
*  `SILO_DBNAME_ALUMNI`
*  `SILO_DBNAME_KB`
*  `SILO_DBNAME_STMATH`
*  `SILO_URL`
*  `SILO_USER`
*  `SILO_PWD`
